
\section{Colors}

An object of the class \ccc{CGAL::Color} is a color available
for drawing operations in many \cgal\ output streams.


Each color is defined by a triple of integers $(r,g,b)$ with
$0\le r,g,b \le 255$, the so-called {\em rgb-value} of the color.
There are a 11 predefined \ccc{Color} constants available:
\ccc{BLACK}, \ccc{WHITE}, \ccc{GRAY}, \ccc{RED}, \ccc{GREEN},
\ccc{DEEPBLUE}, \ccc{BLUE}, \ccc{PURPLE}, \ccc{VIOLET}, \ccc{ORANGE},
and \ccc{YELLOW}.
