The "simplified" graph have been processed to regroup most of the LGPL packages into a "LGPL" node (removed from the simplified graph).

The simplified graph has also been processed by Graphviz tred to remove edges implied by transitivity.

The unfiltered graph has been processed by tred but no other filtering has been applied.

Green nodes are sources: no other package depend on then. Orange nodes are sinks: they only depend on LGPL packages.
